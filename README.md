# Fate-to-Fate-Contagion

This project examines whether the fate of perpetrators in public mass shootings influences the fate of future perpetrators. Using a bivariate Hawkes process model and U.S. public mass shooting data from 1966–2024, the study analyzes contagion dynamics between incidents in which offenders die at the scene and incidents in which they survive. The goal is to better understand how specific behavioral outcomes may spread through media exposure and social influence.


This repository contains the code to the main results, the model validation and the data we used.
